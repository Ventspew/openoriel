Oriel Pulse

Built for play. Still yours.

A darker edition in the same binary — Corner controls, performance limits, local ambience. Same privacy model as Classic.

  • Pulse Corner — launch, savers, ambience, mute, hibernate
  • Data Saver, Lucid Mode, Low Power tightening
  • Mac CPU & RAM governors (real timer throttle)
  • Optional Pulse Dock / home-screen icon

What you get

  • Tabs & library — groups, bookmarks, Reading List, history, downloads that survive relaunch
  • Page tools — find, mute, screenshot, PDF, zoom per site, Reader, translate
  • Profiles — separate cookie jars; Play / Stream / Focus workspaces
  • Extensions — Chrome Web Store, Firefox Add-ons, Safari packages via Oriel Store
  • Password Vault — optional AES-GCM vault with Touch ID; Keychain autofill still works
  • iCloud sync — bookmarks, Reading List, tabs, appearance (not vault secrets)

Oriel Engine on Mac

Oriel Engine is Oriel’s own Blink surface (CEF), open-sourced under Sources/CEF/ and bundled in Mac installers. Smart mode sends tough sites there; Apple hosts stay on WebKit.

iPhone and iPad stay on WebKit only — Apple’s rule, stated plainly.

Oriel Shields

Bundled blocker lists, cosmetics, HTTPS upgrades, tracking-parameter strip, and Fire. Honest about WebKit limits.

Details in docs/PRIVACY.md.
